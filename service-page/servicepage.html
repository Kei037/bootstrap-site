코드절상 무상교환보증 서비스

고객이 무상 보증 상품 사용 중 외부 충격에 의해 코드 절상이 발생한 경우, 정해진 기준에 의거하여 보상 서비스를 제공합니다.

보증 대상

구매일 기준 2년 (구매일자를 증명하지 못할 시 제조일자 기준으로 시행)이내로 마무율이 50% 이내인 제품

보증 범위

사용자 과실 또는 사용자 과살이 아닌 경우로 인한 '코드 절상'인 경우

코드 절상이 발생한 상품만 보상을 실시하며, 코드 절상 발생 시 최초 1회에 한하여 보증
(타이어 장착비, 밸런스, 기타 서비스 비용은 사용자가 부담합니다)

코드 절상이란?

차량운행 중 외부의 돌출물이나 기타 충격에 의해 타이어 옆면이 부풀어 오르는 현상

이미지 area

타이어의 골격을 이루는 카카스 코드가 외부 충격으로 끊어지면, 내부의 높은 공기압이 끊어진 부위를 집중적으로
밀어내기 때문에 타이어 옆면이 혹처럼 부풀어 오르는데, 이것을 코드 절상이라고 합니다.

보증 제외

고의로 흠집이나 변형, 손상을 발생시킨 경우(타인에 의한 경우도 포함)

외부의 화학물질(약품)과의 접촉에 의한 경우

경주용으로 사용한 경우

차량에 적합하지 않은 규격의 타이어를 사용한 경우

잘못된 장착, 부적절한 차륜 정렬 등 차량의 이상에 의한 경우

신차 출고 시 장착된 타이어는 제외

구입일로부터 2년을 초과한 경우

마모율이 50%를 초과한 경우

수리한 흔적이 있는 경우

트레드(노면과 접지되는 면) 부위의 손상 혹은 관통으로 사용이 불가능한 경우

운영기간

2021년 7월 ~ 2022년 6월

보증 대상

구매일 기준 3개월 이내로 마모율이 30% 이내인 제품

보증 범위
